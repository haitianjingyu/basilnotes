# basilnotes在线笔记



basilnotes是一款简洁的在线笔记，利用laravel框架由cxwt工作室开发，该笔记包含写笔记、保存笔记、修改笔记、分类管理、删除恢复、用户注册等功能。可以安装在配置了php运行环境的个人电脑上使用，也可以部署在服务器上供多人使用。此笔记是作者在学习laravel框架时开发出来的，不足之处，敬请指出改正。
